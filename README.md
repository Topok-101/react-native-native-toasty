# react-native-native-toasty

React Native toast notifications with Turbo Native Modules. 

## Installation

```sh
npm install react-native-native-toasty
```

## Usage


```js
import { multiply } from 'react-native-native-toasty';

// ...

const result = multiply(3, 7);
```
